# GardenWise

**Grow Smart. Garden Responsibly.** — React + Vite frontend for plant recommendations, weeds, maps, and weather.

## Stack

- **Frontend:** React 19, TypeScript, Vite, React Router
- **API (optional):** Node.js + Express + PostgreSQL/PostGIS (`server/`) — used for RDS-backed recommendations and weed listings

## Prerequisites

- Node.js 20+ (recommended)
- npm
