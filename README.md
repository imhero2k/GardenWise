# GardenWise

**Grow Smart. Garden Responsibly.** — A React + Vite frontend for plant recommendations, weed identification, maps, and weather insights.

## Stack

- **Frontend:** React 19, TypeScript, Vite, React Router
- **Optional API:** Node.js + Express + PostgreSQL/PostGIS (`server/`) for RDS-backed recommendations and weed listings

## Prerequisites

- Node.js 20+ recommended
- npm

## Setup

```bash
npm install
